# 주요 조치

방사형에서 기존 키 바인딩(바닐라 또는 모드)을 실행하려면 키 동작을 사용하세요.

## 필드

- **제목**: 편집기/방사형 라벨에 표시되는 내용입니다.
- **참고**: 선택적 도우미 텍스트입니다.
- **매핑 이름**: 키 매핑 ID 또는 라벨입니다.
- **배달**: `AUTO`, `INPUT` 또는 `TICK`.
- **토글**: 탭하는 대신 키 상태를 전환합니다.
- **아이콘**: 아이콘 상자를 클릭하여 아이콘 선택기를 엽니다.

## 권장 설정

1. `Pick from Keybinds`을 클릭하세요.
2. 선택기에서 바인딩을 선택합니다(손으로 입력하는 것보다 안전함).
3. 특별한 사유가 없는 한 `AUTO`으로 배송을 유지하세요.
4. 저장합니다.

## 전달 모드

| 모드 | 그것이 하는 일 | 사용 시기 |
|---|---|---|
| `AUTO` | 자동으로 최적의 경로 선택 | 거의 모든 사람을 위한 기본값 |
| `INPUT` | 입력 파이프라인을 통해 키 누름/해제 주입 | `AUTO`에서 바인딩이 응답하지 않는 경우 |
| `TICK` | 키 상태 업데이트를 통해 키 다운/업 설정 | 삽입하기 어려운 키에 대한 유용한 대체 |

## 토글

- `OFF`: 방사형 사용당 탭 1개.
- `ON`: 사용할 때마다 키를 아래/위로 뒤집습니다.

대상 키바인드의 작동 방식에 따라 스프린트/스니크 스타일 토글과 같은 작업에 유용합니다.

## 일반적인 문제

- **아무 일도 일어나지 않습니다.** 매핑 ID를 확인하거나 키 바인딩 목록에서 다시 선택하세요.
- **잘못된 키 바인딩 트리거:** 입력한 이름이 다른 매핑 라벨과 일치합니다. 선택기를 사용하십시오.
- **싱글 플레이어에서는 작동하지만 서버에서는 작동하지 않습니다.** 대상 작업은 서버에서 제한될 수 있습니다.

???+ info "심층 분석: 키 전달 작동 방식"
    내부적으로 EZ Actions는 다음을 통해 매핑 이름을 확인합니다.

1. 정확한 번역 키 일치
    2. 정확한 현지화된 라벨 일치
    3. 대체에는 일치 항목이 포함됩니다.

수정자가 필요한 키(Ctrl/Shift/Alt)의 경우 추가 주입 논리를 사용합니다. `AUTO`은 키가 바인딩되었는지, 스캔코드가 있는지, 수정자가 필요한지에 따라 전략을 전환할 수 있습니다.
