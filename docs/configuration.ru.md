# Конфигурация

Откройте конфигурацию из редактора меню с помощью кнопки `Config`.

Пользовательский интерфейс конфигурации состоит из трех разделов:

- Общий
- Анимации
- Дизайн

## Общий

| Настройка | Тип | Диапазон/значения | По умолчанию |
|---|---|---|---|
| Двигайтесь при радиальном открытии | логическое | ВКЛ/ВЫКЛ | ВКЛ |
| Показать метку при наведении | логическое | ВКЛ/ВЫКЛ | ВКЛ |
| Редактор команд Видимые линии | интервал | 1-20 | 5 |

## Анимации

| Настройка | Тип | Диапазон/значения | По умолчанию |
|---|---|---|---|
| Анимация включена | логическое | ВКЛ/ВЫКЛ | ВКЛ |
| Анимация открытия/закрытия | логическое | ВКЛ/ВЫКЛ | ВКЛ |
| Анимация при наведении | логическое | ВКЛ/ВЫКЛ | ВКЛ |
| Открытый стиль | перечисление | `WIPE`, `FADE`, `NONE` | `WIPE` |
| Направление | перечисление | `CW`, `CCW` | `CW` |
| Стиль при наведении | перечисление | `FILL_SCALE`, `FILL_ONLY`, `SCALE_ONLY`, `NONE` | `FILL_SCALE` |
| Процент роста при наведении | двойной | 0,0-0,5 | 0,05 |
| Продолжительность открытия/закрытия | целое (мс) | 0-2000 | 125 |

## Дизайн

| Настройка | Тип | Диапазон/значения | По умолчанию |
|---|---|---|---|
| Мертвая зона | интервал | 0-90 | 18 |
| Внешний радиус | интервал | 24-512 | 72 |
| Толщина кольца | интервал | 6-256 | 28 |
| Начальный порог шкалы | интервал | 0-128 | 8 |
| Масштаб по каждому элементу | интервал | 0-100 | 6 |
| Срез разрыв | целое (градусы) | 0-12 | 0 |
| Стиль дизайна | перечисление | `SOLID`, `SEGMENTED`, `OUTLINE`, `GLASS` | `SOLID` |
| Цвет кольца | ARGB-интервал | 32-битное целое число со знаком | `0xAA000000` |
| Цвет при наведении | ARGB-интервал | 32-битное целое число со знаком | `0xFFF20044` |
| Цвет границы | ARGB-интервал | 32-битное целое число со знаком | `0x66FFFFFF` |
| Цвет текста | ARGB-интервал | 32-битное целое число со знаком | `0xFFFFFFFF` |

## Предварительный просмотр

`Preview` открывает радиальный экран предварительного просмотра и зацикливает анимацию с паузой в 1 секунду между циклами.

Используйте это для настройки визуальных настроек перед фиксацией.

## Сохранить поведение

- Save записывает значения сразу в файлы конфигурации.
- При сохранении экран возвращается к экрану родительского редактора.

## Конфигурационные файлы

- `config/ezactions/general-client.toml`
- `config/ezactions/anim-client.toml`
- `config/ezactions/design-client.toml`

???+ информация "Глубокое погружение: цветовой формат"
    Цвета хранятся как подписанные 32-битные целые числа ARGB.

Пример:

- `0xAARRGGBB`
    - альфа `AA`, затем красный `RR`, зеленый `GG`, синий `BB`

Отрицательные десятичные числа являются нормальными для непрозрачных значений ARGB.
