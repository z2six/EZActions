# メインメニューエディタのGUI

メニュー エディタは、ラジアルを構築するためのコントロール センターです。

`Open editor` キーバインドを使用して開きます。

## レイアウト

- **左側のパネル:** アクションとバンドルを作成/編集/削除します。
- **右パネル:** 現在のページのリスト (ルートまたは現在のバンドル)。
- **左上のフィルター:** タイトル、メモ、およびアクション タイプのテキストによってフィルターします。
- **左下:** インポート/エクスポート、構成、閉じる。

## リストの対話

### マウス

- **アイテムを左クリック:** 選択します。
- **項目を左クリックでドラッグ:** 現在のページ内で並べ替えます。
- **LMB でアイテムをバンドル行にドラッグします:** アイテムをそのバンドルに移動します。
- **LMB で項目を後行にドラッグします:** 項目を親/ルートに移動します。
- **バンドル行の RMB:** そのバンドルを開きます。

### キーボード

- `Ctrl + F` はフィルター ボックスに焦点を当てます。
- `Enter` は選択した行を編集します。
- `Delete` または `Backspace` は、選択した行を削除します。
- `Up Arrow` は、選択した項目を上に移動します。
- `Down Arrow` は、選択した項目を下に移動します。

!!!ヒント
    あいまいな並べ替えを避けるため、フィルタ テキストがアクティブな間はキーボードの上下移動は無効になります。

## 表示される行の種類

- 通常のアイテム行 (アクションまたはバンドル)
- ブレッドクラム行 (`root/.../bundle`)
- `Back to root`
- `Back to <parent>`

## ロックされたエントリ

ロックされたエントリは、ゲーム内の削除操作から保護されます。

- GUI 削除では削除できません。
- ロックを優先する API 削除呼び出しでは削除できません。
- `config/ezactions/menu.json` を手動で編集することで削除できます。

## 高速編集のためのヒント

- 放射状のラベルをきれいにするには、短いタイトルを使用します。
- 共有ユーティリティ アクションをバンドルに配置します (ルートの混乱を軽減するため)。
- メニューが膨大になる場合は、フィルター ボックスを素早い入力検索として使用します。

???+ 情報「詳細: ドラッグ アンド ドロップの動作」
    - 並べ替えは挿入ロジックを使用します (単純なスワップではありません)。
    - バンドルにドロップすると、そのバンドルの子リストに追加されます。
    - 親/ルートにドロップすると、自動ジャンプではなく現在のエディター ビューが維持されます。
    - 成功したすべての移動/削除操作は、直ちにディスクに保存されます。
