# 아이템 장착 액션

아이템 장착을 사용하면 장비 스냅샷을 저장하고 인벤토리에서 일치하는 아이템을 다시 장착할 수 있습니다.

## 대상 슬롯

다음 중 하나를 할당할 수 있습니다.

- 직접
- 오프핸드
- 헬멧
- 흉갑
- 레깅스
- 부츠

편집기에서 슬롯이 비어 있는 경우 EZ Actions는 실행 중에 해당 슬롯을 건너뜁니다.

## 만드는 방법

1. `Add Item Equip`을 클릭하세요.
2. 제목/메모/아이콘을 설정하세요.
3. 소스 그리드의 항목을 대상 슬롯으로 드래그합니다.
4. 저장합니다.

### 소스 그리드 포함

- 현재 손에 쥔 물건
- 장착된 갑옷
- 주요 인벤토리
- 단축바

## 실행 중

방사형에서 작업을 트리거하는 경우:

1. EZ Actions는 구성된 각 대상 슬롯을 확인합니다.
2. 대상이 이미 녹음된 항목과 일치하는 경우 건너뜁니다.
3. 그렇지 않은 경우 가장 일치하는 소스 스택을 찾습니다.
4. 아이템을 대상 슬롯으로 교체합니다.

슬롯별로 처리하고 부분적인 성공을 허용합니다.

## 중요한 매칭 규칙

- 일치는 전체 스택 스냅샷 서명(NBT 및 메타데이터)을 기반으로 하며 개수는 무시됩니다.
- 일치하는 스택이 여러 개인 경우 가장 높은 개수의 스택을 선택합니다.

## 메인핸드 규칙

`Mainhand`은 트리거 시간에 현재 선택된 핫바 슬롯을 의미합니다.

## 압력을 받는 행동

- 실행 중인 아이템 장착 작업을 두 번째로 실행하면 기존 작업이 취소되고 교체됩니다.
- 실행이 백그라운드 틱에서 실행되는 동안 이동 및 게임플레이 입력은 활성 상태로 유지됩니다.

## 편집기의 빠른 컨트롤

- 소스에서 대상 슬롯으로 LMB 드래그: 항목 할당.
- 대상 슬롯의 RMB: 할당을 지웁니다.
- `Refresh Items`: 현재 플레이어 인벤토리 상태에서 소스 목록을 다시 작성합니다.

???+ info "Deep dive: 슬롯 실행 순서"
    현재 실행 순서는 다음과 같습니다.

1. 헬멧
    2. 흉갑
    3. 레깅스
    4. 부츠
    5. 오프핸드
    6. 메인핸드

빈 대상과 누락된 소스 일치는 건너뛰며 심각한 오류로 처리되지 않습니다.
