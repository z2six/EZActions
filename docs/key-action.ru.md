# Ключевое действие

Используйте действие клавиши, чтобы активировать существующую привязку клавиш (ванильную или модифицированную) из радиальной области.

## Поля

- **Название**: то, что вы видите в редакторе/радиальной метке.
- **Примечание**: необязательный вспомогательный текст.
- **Имя сопоставления**: идентификатор или метка сопоставления клавиш.
- **Доставка**: `AUTO`, `INPUT` или `TICK`.
- **Переключение**: переключение состояния клавиши вместо нажатия.
- **Значок**: щелкните поле значка, чтобы открыть средство выбора значков.

## Рекомендуемая настройка

1. Нажмите `Pick from Keybinds`.
2. Выберите привязку из средства выбора (безопаснее, чем вводить вручную).
3. Продолжайте доставку `AUTO`, если у вас нет особой причины.
4. Сохраните.

## Режимы доставки

| Режим | Что он делает | Когда использовать |
|---|---|---|
| @@КОД0@@ | Автоматически выбирает лучший путь | По умолчанию почти для всех |
| `INPUT` | Вводит нажатие/отпускание клавиши через входной конвейер | Если привязка не отвечает в `AUTO` |
| `TICK` | Устанавливает нажатие/поднятие клавиши посредством обновлений состояния клавиш | Полезный запасной вариант для трудно внедряемых ключей |

## Переключать

- `OFF`: одно нажатие на одно радиальное использование.
- `ON`: переворачивает клавишу вниз/вверх при каждом использовании.

Полезно для таких действий, как переключение стиля спринта/скрытности, в зависимости от того, как ведет себя целевая комбинация клавиш.

## Распространенные проблемы

- **Ничего не происходит:** проверьте идентификатор сопоставления или еще раз выберите из списка привязок клавиш.
- **Неправильные триггеры привязки клавиш:** введенное вами имя соответствует другой метке сопоставления; используйте сборщик.
- **Работает в одиночной игре, но не на сервере:** целевое действие может быть ограничено сервером.

???+ информация "Глубокое погружение: как работает доставка ключей"
    Внутри EZ Actions разрешает имена сопоставлений следующим образом:

1. точное совпадение ключей перевода
    2. точное соответствие локализованной этикетки
    3. резервный вариант содержит совпадение

Для клавиш, требующих модификатора (Ctrl/Shift/Alt), используется дополнительная логика внедрения. `AUTO` может переключать стратегию в зависимости от того, привязан ли ключ, имеет ли он скан-код или требует модификаторов.
