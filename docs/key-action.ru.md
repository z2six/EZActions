# Key Action

`Key Action` запускает существующий keybind (ванильный или модовый) из радиала.

## Поля

- **Title**
- **Note**
- **Mapping Name**
- **Delivery**: `AUTO`, `INPUT`, `TICK`
- **Toggle**
- **Icon**

## Рекомендуемый процесс

1. Нажми `Pick from Keybinds`.
2. Выбери биндинг из списка.
3. Оставь `AUTO`, если нет особой причины менять.
4. Сохрани.

## Delivery режимы

| Mode | Что делает | Когда использовать |
|---|---|---|
| `AUTO` | Авто-выбор лучшего способа | По умолчанию |
| `INPUT` | Инъекция key press/release в input pipeline | Если `AUTO` не срабатывает |
| `TICK` | Управление key state по тикам | Резервный вариант |

## Toggle

- `OFF`: один tap на использование.
- `ON`: переключение down/up на каждое использование.

## Частые проблемы

- **Ничего не происходит:** проверь mapping id, попробуй выбрать снова через picker.
- **Срабатывает не тот keybind:** ручной ввод дал неверный match, используй picker.
- **Работает в одиночке, но не на сервере:** действие может блокироваться сервером.
