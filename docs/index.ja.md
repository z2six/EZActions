# EZ アクション Wiki

!!!ヒント「言語・语言」
    Wiki 言語を変更するには、右上のナビゲーション バーにある **言語スイッチャー** アイコンを使用します。

    右上角ナビゲーション中の**発言切替え**図标来 Wiki 発言を使用してください。

EZ Actions はクライアント側の Minecraft Mod で、常に使用するアクションに高速な放射状メニューを提供します。

これを「戦闘 + ユーティリティ クイック ホイール」と考えてください。キーを 1 回押して、1 回フリックするだけで完了します。

!!!警告「バージョンスコープ」
    この Wiki は **EZ Actions 2.0.0.0** 向けに書かれています。

    新しいバージョンを使用している場合は、一部の機能と UI の詳細が変更されている可能性があります。

???+情報「TLDR」
    - **キー アクション**、**コマンド アクション**、**アイテム装備アクション**、**バンドル**を使用して独自のラジアル メニューを作成します。
    - 色/デザイン/アニメーション構成でスタイルを設定します。
    - 共有とバックアップのためのインポート/エクスポート メニュー JSON。
    - Mod 開発者は API を通じてすべてを制御できます (一時的なランタイム ラジアルを含む)。

## EZ アクションでできること

- バニラまたは改造されたキーバインドをトリガーします。
- 単一行または複数行のコマンドを実行します。
- 正確なアイテムマッチングを使用して、記録されたギアセットを装備します（NBTを含む）。
- アクションをネストされたバンドルに整理します。
- バンドル キーバインド経由でバンドルにアクセスできるようにしながら、ルートからバンドルを非表示にします。
- `config/ezactions/icons` からカスタム アイコンを追加します。
- ドラッグ/ドロップやキーボード ショートカットを使用してゲーム内でメニューを作成/編集します。
- 他の MOD がパブリック API を通じて EZ アクションを駆動できるようにします。

## この Wiki の対象者

ほとんどのページには、展開可能な「詳細」セクションが表示されます。実際の流れだけを知りたい場合はスキップしてください。

## クイックスタート

1. 以下のキーバインドを設定します。
   - `Open radial menu`
   - `Open editor`
2. エディターを開き、最初のアクションを追加します。
3. ゲーム内でラジアル キーを押したまま、スライスの上で放して実行します。
4. [構成] 画面でビジュアルを調整します。

??? 「テクニカルノート」
    EZ Actions は完全にクライアント側です。サーバーのインストールは必要ありません。

    アクションは、サーバーが許可するもの (コマンド権限など) に依存します。
