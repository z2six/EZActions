# Bundles

`Bundle`은 Radial 트리 안의 폴더/카테고리입니다.

## Bundle 필드

- **Title**
- **Note**(선택)
- **Icon**
- **Hide from main radial**
- **Enable keybind**

## Bundle keybind

`Enable keybind`를 켜면 해당 Bundle 전용 keybind가 등록됩니다.

!!! warning "재시작 필요"
    Bundle keybind 등록은 클라이언트 재시작 후 적용됩니다.

## Hide from main radial

활성화 시:

- Root 페이지에서 숨김
- 메뉴 모델에는 유지
- API 또는 Bundle keybind로 접근 가능

## locked Bundle

`locked`는 게임 내 삭제를 막습니다.

- GUI 삭제 불가
- JSON 수동 편집으로는 삭제 가능
