## Supported Platforms

| Loader     | Minecraft Version | Notes |
|-------------|------------------|--------|
| **NeoForge** | 1.21.1 | Fully supported |
| **Forge**    | 1.20.1 | Fully supported |
| *(Planned)*  | Fabric, Quilt | Future support planned |

EZActions is designed using a **Multiloader architecture**, meaning most of its logic is shared between versions. Each loader-specific implementation uses a lightweight platform helper for full compatibility.

---

## About

EZActions is a **client-side utility mod** that introduces a circular (radial) quick-access menu. Players can map any in-game keybind, mod keybind, or console command into an intuitive layout, ideal for complex modpacks or fast gameplay.

Use it to:
- Group your most-used commands or abilities.
- Trigger actions with one motion instead of multiple key presses.
- Organize actions into nested categories (bundles).
- Create custom menu layouts that persist between sessions.
