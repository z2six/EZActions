# EZ 액션 위키

!!! Tip "언어 / 语言"
    위키 언어를 변경하려면 오른쪽 상단 탐색 표시줄에 있는 **언어 전환기** 아이콘을 사용하세요.

    使用右上角导航栏中的**语言切换**图标来切换 Wiki 语言。

EZ Actions는 항상 사용하는 작업에 대한 빠른 방사형 메뉴를 제공하는 클라이언트측 Minecraft 모드입니다.

"전투 + 유틸리티 퀵 휠"이라고 생각하세요. 키를 한 번 누르고, 가볍게 치면 완료됩니다.

!!! "버전 범위" 경고
    이 위키는 **EZ Actions 2.0.0.0**용으로 작성되었습니다.

    최신 버전을 사용하는 경우 일부 기능 및 UI 세부정보가 변경되었을 수 있습니다.

???+ 정보 "TLDR"
    - **주요 작업**, **명령 작업**, **아이템 장착 작업** 및 **번들**을 사용하여 자신만의 방사형 메뉴를 구축하세요.
    - 색상/디자인/애니메이션 구성으로 스타일을 지정하세요.
    - 공유 및 백업을 위한 JSON 가져오기/내보내기 메뉴.
    - 모드 개발자는 API를 통해 모든 것을 제어할 수 있습니다(임시 런타임 방사형 포함).

## EZ Action으로 할 수 있는 작업

- 바닐라 또는 수정된 키바인딩을 실행합니다.
- 단일 또는 여러 줄 명령을 실행합니다.
- 정확한 아이템 매칭을 통해 기록된 장비 세트를 장착하세요(NBT 포함).
- 작업을 중첩된 번들로 구성합니다.
- 번들 키 바인딩을 통해 액세스할 수 있도록 유지하면서 루트에서 번들을 숨깁니다.
- `config/ezactions/icons`의 맞춤 아이콘을 추가하세요.
- 드래그/드롭 및 키보드 단축키를 사용하여 게임 내 메뉴를 구축/편집합니다.
- 다른 모드가 공개 API를 통해 EZ Action을 구동하도록 허용합니다.

## 이 위키는 누구를 위한 것인가요?

- 소스 코드를 읽지 않고도 명확한 설정 가이드를 원하는 플레이어.
- 고급 동작 세부정보를 원하는 고급 사용자.
- 완전한 API 문서를 원하는 모드팩 제작자 및 모드 개발자.

대부분의 페이지에는 확장 가능한 '심층 분석' 섹션이 표시됩니다. 단지 실용적인 흐름을 원한다면 건너뛰세요.

## 빠른 시작

1. 다음에 대한 키바인딩을 설정합니다:
   - `Open radial menu`
   - `Open editor`
2. 편집기를 열고 첫 번째 작업을 추가합니다.
3. 게임 내에서 방사형 키를 누른 상태에서 슬라이스 위에 놓아 실행합니다.
4. 구성 화면에서 시각적 요소를 조정합니다.

전체 문서를 보려면 왼쪽 탐색 메뉴를 사용하세요.

??? "기술 노트"
    EZ Actions는 완전히 클라이언트 측입니다. 서버 설치가 필요하지 않습니다.

    작업은 여전히 ​​서버에서 허용하는 사항(예: 명령 권한)에 따라 달라집니다.
