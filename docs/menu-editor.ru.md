# Графический интерфейс редактора главного меню

Редактор меню — это ваш центр управления построением радиала.

Откройте его с помощью сочетания клавиш `Open editor`.

## Макет

- **Левая панель:** создание, редактирование и удаление действий и пакетов.
- **Правая панель:** текущий список страниц (корневой или текущий пакет).
– **Фильтр вверху слева:** фильтрация по заголовку, примечанию и тексту типа действия.
– **Слева внизу:** импорт/экспорт, настройка, закрытие.

## Добавить кнопки

- `Add Key Action`
- `Add Command`
- `Add Item Equip`
- `Add Bundle`

## Взаимодействие со списком

### Мышь

- **ЛКМ на предмете:** выбрать.
- **Перетаскивание элемента ЛКМ:** изменение порядка внутри текущей страницы.
- **Перетащите элемент ЛКМ в строку пакета:** переместите элемент в этот пакет.
- **ЛКМ перетащите элемент на задние строки:** переместите элемент в родительский/корневой каталог.
- **ПКМ по строке пакета:** откройте этот пакет.

### Клавиатура

- `Ctrl + F` фокусирует поле фильтра.
- `Enter` редактирует выбранную строку.
- `Delete` или `Backspace` удаляет выбранную строку.
- `Up Arrow` перемещает выбранный элемент вверх.
- `Down Arrow` перемещает выбранный элемент вниз.

!!! чаевые
    Перемещение клавиатуры вверх/вниз отключено, пока активен фильтр текста, чтобы избежать неоднозначного изменения порядка.

## Типы строк, которые вы увидите

- Обычные строки предметов (действия или пакеты)
- Строка навигации (`root/.../bundle`)
- `Back to root`
- `Back to <parent>`

## Заблокированные записи

Заблокированные записи защищены от операций удаления в игре.

- Вы не можете удалить их с помощью GUI-удаления.
- Вы не можете удалить их с помощью вызовов удаления API, которые учитывают блокировку.
- Их по-прежнему можно удалить, отредактировав `config/ezactions/menu.json` вручную.

## Советы по быстрому редактированию

– Используйте короткие заголовки для более понятных радиальных меток.
- Поместите общие служебные действия в пакеты (для меньшего беспорядка).
- Используйте поле фильтра для быстрого поиска по типу, когда ваше меню становится огромным.

???+ информация «Глубокое погружение: поведение перетаскивания»
    - Reorder uses insertion logic (not simple swap).
    - Dropping into a bundle appends to that bundle's child list.
    - Dropping to parent/root keeps your current editor view instead of auto-jumping.
    - All successful move/remove operations persist immediately to disk.
