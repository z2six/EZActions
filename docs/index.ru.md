# EZ Actions Wiki

!!! tip "Language"
    Use the **language switcher** icon in the top-right navigation bar to change wiki language.

EZ Actions — это клиентский мод Minecraft с быстрым радиальным меню для часто используемых действий.

Проще: зажал кнопку, выбрал сектор, отпустил — действие выполнено.

!!! warning "Область версии"
    Эта wiki написана для **EZ Actions 2.0.0.0**.

    В более новых версиях часть функций и UI может отличаться.

???+ info "TLDR"
    - Собирай свой радиал из **Key Action**, **Command Action**, **Item Equip Action** и **Bundle**.
    - Настраивай цвета/дизайн/анимации.
    - Импортируй/экспортируй JSON для обмена и бэкапов.
    - Разработчики могут управлять всем через API.

## Что умеет EZ Actions

- Триггерить ванильные и модовые keybind.
- Выполнять одиночные и многострочные команды.
- Переэкипировать наборы по точному совпадению предмета (включая NBT).
- Группировать действия во вложенные Bundle.
- Скрывать Bundle из root-колеса, но оставлять доступ через keybind.
- Загружать кастомные иконки из `config/ezactions/icons`.
- Редактировать меню в игре (drag/drop + хоткеи).
- Давать другим модам контроль через публичный API.

## Для кого эта wiki

- Для игроков, которым нужен понятный гайд без чтения исходников.
- Для продвинутых пользователей, которым важны детали поведения.
- Для авторов сборок и разработчиков модов.

## Быстрый старт

1. Назначь keybind для:
   - `Open radial menu`
   - `Open editor`
2. Открой редактор и добавь первое действие.
3. В игре удерживай кнопку радиала и отпусти на нужном секторе.
4. Подкрути визуал в разделе Config.
