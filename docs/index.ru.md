EZ Actions — это клиентский мод Minecraft, который предоставляет вам быстрое круговое меню для действий, которые вы используете постоянно.

Думайте об этом как о «быстром колесе боя + утилиты»: одно удержание клавиши, одно движение — и готово.

!!! предупреждение «Область версии»
    Эта вики написана для **EZ Actions 2.0.0.0**.

    Если вы используете более новую версию, некоторые функции и детали пользовательского интерфейса могли измениться.

???+ информация "TLDR"
    - Создайте свое собственное круговое меню с помощью **Ключевых действий**, **Командных действий**, **Действий по экипировке предметов** и **Наборов**.
    - Оформите его с помощью конфигураций цвета/дизайна/анимации.
    - Меню импорта/экспорта JSON для совместного использования и резервного копирования.
    - Разработчики модов могут контролировать все через API (включая временные радиалы времени выполнения).

## Что могут действия EZ

- Запускайте ванильные или модифицированные сочетания клавиш.
- Запуск одно- или многострочных команд.
- Используйте записанные наборы снаряжения, используя точное соответствие предметов (включая NBT).
- Организуйте действия во вложенные пакеты.
- Скрыть пакеты от root, сохраняя при этом доступ к ним через привязку клавиш пакета.
- Добавьте собственные значки из `config/ezactions/icons`.
- Создавайте/редактируйте внутриигровые меню с помощью перетаскивания и сочетаний клавиш.
- Позвольте другим модам управлять EZ Actions через общедоступный API.

## Для кого предназначена эта вики

- Игроки, которым нужно четкое руководство по установке без чтения исходного кода.
- Опытные пользователи, которым нужны расширенные сведения о поведении.
- Создатели модпаков и разработчики модов, которым нужна полная документация по API.

На большинстве страниц вы увидите расширяемые разделы «глубокого погружения». Пропустите их, если вам просто нужен практический ход.

## Быстрый старт

1. Установите привязки клавиш для:
   - `Open radial menu`
   - `Open editor`
2. Откройте редактор и добавьте свое первое действие.
3. Удерживайте радиальную клавишу в игре и отпустите ее над фрагментом, чтобы выполнить действие.
4. Настройте визуальные эффекты на экране конфигурации.

## Навигация

Используйте левую навигацию для просмотра полных документов:

- Графический интерфейс редактора главного меню
- Ключевое действие
- Действие команды
- Действие при экипировке предмета
- Пакеты
- Импорт Экспорт
- Конфигурация
- API

??? «Техническая записка»
    EZ Actions полностью клиентский. Не требует установки сервера.

    Действия по-прежнему зависят от того, что разрешает сервер (например, разрешения на выполнение команд).
