# Предмет Экипировка Действие

Item Equip позволяет вам сохранить снимок снаряжения и повторно экипировать соответствующие предметы из вашего инвентаря.

## Целевые слоты

Вы можете назначить любое из них:

- Основная рука
- Левая рука
- Шлем
- Нагрудник
- Леггинсы
- Сапоги

Если слот в редакторе оставлен пустым, EZ Actions пропускает его во время выполнения.

## Как создать

1. Нажмите `Add Item Equip`.
2. Установите заголовок/заметку/значок.
3. Перетащите элементы из исходной сетки в целевые слоты.
4. Сохраните.

### Исходная сетка включает в себя

- текущий дополнительный предмет
- экипированная броня
- основной инвентарь
- горячая панель

## Во время выполнения

Когда вы запускаете действие радиально:

1. EZ Actions проверяет каждый настроенный целевой слот.
2. Если цель уже соответствует записанному элементу, она пропускается.
3. Если нет, он находит наиболее подходящий исходный стек.
4. Он перемещает предметы в целевой слот.

Он обрабатывает слот за слотом и обеспечивает частичный успех.

## Важные правила сопоставления

- Сопоставление основано на сигнатуре полного снимка стека (NBT и метаданные), без учета количества.
- Если существует несколько совпадающих стопок, выбирается тот, у которого наибольшее количество.

## Правило основной руки

`Mainhand` означает выбранный вами в данный момент слот панели быстрого доступа во время триггера.

## Поведение под давлением

— Если вы активируете второе действие «Снаряжение предмета», пока оно выполняется, старое действие отменяется и заменяется.
- Вводы движения и игрового процесса остаются активными, пока выполнение выполняется в фоновом режиме.

## Быстрое управление в редакторе

- Перетаскивание ЛКМ из исходного слота в целевой: назначить предмет.
- ПКМ по целевому слоту: четкое назначение.
- `Refresh Items`: перестроить список источников из текущего состояния инвентаря игрока.
