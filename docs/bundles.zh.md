# 组合（Bundle）

组合是径向树中的“文件夹 / 分类”。

你可以按场景组织操作：战斗、建造、工具、角色扮演等。

## 组合字段

- **标题**（也用于内部标识）
- **备注**（可选）
- **图标**
- **在主径向菜单中隐藏**
- **启用按键绑定**

## 组合按键绑定

开启 `Enable keybind` 后，EZ Actions 会为该组合注册独立按键绑定。

!!! warning "需要重启"
    组合按键绑定注册会在下次客户端重启后生效。

    当需要重启时，EZ Actions 会给出客户端提示（包括 API 创建的组合）。

## 在主径向菜单中隐藏

开启后：

- 组合不会显示在根径向页面。
- 组合仍然存在于菜单模型中。
- 仍可通过 API 或组合按键打开。

适合做“高级页面”，避免根页面过于拥挤。

## 嵌套组合

组合内可包含：

- 按键动作
- 命令动作
- 装备切换动作
- 更多组合

## 最佳实践

- 根页面保持精简，只放高频操作。
- 低频操作下沉到更深层组合。
- 给组合使用清晰图标和短标题。

## 锁定组合

组合可标记为 `locked`（通常由 API 或 JSON 设置）。

- 游戏内删除路径不会删除它。
- 手动改 JSON 仍可移除。

???+ info "深入说明：标识与唯一性"
    在编辑器工作流中，组合标题会作为组合 ID 使用。

    重复组合名会导致歧义，因此编辑器会阻止重复标题 / ID 保存。
