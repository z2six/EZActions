# Keybinds

EZActions adds several keybinds under **Category: EZActions**.

You can customize these from **Options → Controls → Keybinds**.

---

| Keybind | Default | Description |
|----------|----------|-------------|
| **Open EZActions** | ` (backtick) | Opens the radial menu |
| **Open Editor** | Unbound | Opens the Menu Editor GUI |
