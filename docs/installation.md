# Installation

EZActions can be installed for both **Forge (1.20.1)** and **NeoForge (1.21.1)**.

---

## Requirements

| Component | Minimum Version | Notes |
|------------|----------------|-------|
| Minecraft | 1.20.1 (Forge) / 1.21.1 (NeoForge) | Client-side only |
| Java | 21 | Required for both builds |
| Loader | Forge or NeoForge | Match the `.jar` you download |
