# 主菜单编辑器界面

菜单编辑器是你构建径向菜单的控制中心。

通过 `Open editor` 按键打开。

## 布局

- **左侧面板：** 创建 / 编辑 / 删除动作与组合。
- **右侧面板：** 当前页面列表（根页面或当前组合）。
- **左上筛选框：** 按标题、备注、动作类型筛选。
- **左下区域：** 导入 / 导出、配置、关闭。

## 添加按钮

- `Add Key Action`
- `Add Command`
- `Add Item Equip`
- `Add Bundle`

## 列表交互

### 鼠标

- **左键点击条目：** 选中。
- **左键拖拽条目：** 在当前页面内重排。
- **拖到组合行上：** 将条目移入该组合。
- **拖到返回行上：** 将条目移到父级 / 根级。
- **右键组合行：** 打开该组合。

### 键盘

- `Ctrl + F` 聚焦筛选框。
- `Enter` 编辑当前选中行。
- `Delete` 或 `Backspace` 删除当前选中行。
- `上方向键` 上移选中条目。
- `下方向键` 下移选中条目。

!!! tip
    当筛选框有文本时，上下移动会被禁用，以避免筛选状态下的重排歧义。

## 你会看到的行类型

- 普通条目行（动作或组合）
- 面包屑行（`root/.../bundle`）
- `Back to root`
- `Back to <parent>`

## 锁定条目（Locked）

锁定条目会受到保护，无法在游戏内直接删除。

- 不能通过 GUI 删除。
- 不能被遵守锁定规则的 API 删除调用移除。
- 仍可通过手动编辑 `config/ezactions/menu.json` 删除。

## 高效编辑建议

- 标题尽量简短，径向标签更清晰。
- 把通用工具动作放进组合，减少根菜单拥挤。
- 菜单很大时，优先用筛选框做快速定位。

???+ info "深入说明：拖拽行为"
    - 重排使用“插入逻辑”，不是简单交换。
    - 丢进组合时会追加到该组合子列表末尾。
    - 丢到父级 / 根级后，编辑器不会自动跳页，保持当前视图。
    - 成功移动 / 删除后会立即持久化到磁盘。
