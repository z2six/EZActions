EZ Actions 是一个客户端 Minecraft 模组，为您提供一个快速的径向菜单，用于您一直使用的操作。

把它想象成你的“战斗+实用快轮”：按住键，轻弹一下，完成。

!!!警告“版本范围”
    本 wiki 是为 **EZ Actions 2.0.0.0** 编写的。

    如果您使用的是较新版本，某些功能和 UI 详细信息可能已更改。

？？？+信息“TLDR”
    - 使用**关键操作**、**命令操作**、**物品装备操作**和**捆绑包**构建您自己的径向菜单。
    - 使用颜色/设计/动画配置对其进行设计。
    - 导入/导出菜单 JSON 用于共享和备份。
    - Mod 开发人员可以通过 API 控制一切（包括临时运行时径向）。

## EZ Action 可以做什么

- 触发原版或修改后的按键绑定。
- 运行单行或多行命令。
- 使用精确的物品匹配来装备记录的装备组（包括NBT）。
- 将操作组织到嵌套的包中。
- 从根目录隐藏捆绑包，同时保持通过捆绑包键绑定可访问它们。
- 添加来自`config/ezactions/icons`的自定义图标。
- 使用拖放和键盘快捷键在游戏中构建/编辑菜单。
- 让其他模组通过公共 API 驱动 EZ Actions。
