# EZ Actions Wiki

!!! tip "语言切换 / Language"
    使用右上角导航栏中的**语言切换**图标来切换 Wiki 语言。

    Use the **language switcher** icon in the top-right navigation bar.

EZ Actions 是一个纯客户端 Minecraft 模组，提供一个高效率的径向菜单，专门放你最常用的操作。

可以把它当成你的“战斗 + 工具快捷轮盘”：按住键，轻轻一甩，执行完成。

!!! warning "版本范围"
    本 Wiki 基于 **EZ Actions 2.0.0.0** 编写。

    如果你使用的是更新版本，部分功能和界面细节可能会不同。

???+ info "TLDR"
    - 你可以用 **按键动作**、**命令动作**、**装备切换动作**、**组合** 来构建自己的径向菜单。
    - 可在配置里调整颜色 / 设计 / 动画。
    - 支持导入 / 导出 JSON，方便分享和备份。
    - 模组开发者可通过 API 完整控制（包含临时运行时径向菜单）。

## EZ Actions 能做什么

- 触发原版或其他模组的按键绑定。
- 执行单行或多行命令。
- 通过精确匹配（包含 NBT）切换你记录过的装备。
- 用嵌套组合整理操作结构。
- 将组合从根菜单隐藏，但仍可通过组合按键打开。
- 从 `config/ezactions/icons` 加载自定义图标。
- 在游戏内拖拽、排序、快捷键编辑菜单。
- 让其他模组通过公开 API 驱动 EZ Actions。

## 快速开始

1. 先设置以下按键：
   - `Open radial menu`
   - `Open editor`
2. 打开编辑器并添加第一个动作。
3. 游戏内按住径向菜单键，移动到扇区后松开执行。
4. 在配置界面微调视觉效果。

??? "技术说明"
    EZ Actions 完全是客户端模组，不要求服务器安装。

    但动作是否真正生效仍取决于服务器规则（例如命令权限）。
