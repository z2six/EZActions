# 메인 메뉴 편집기 GUI

메뉴 편집기는 방사형을 만들기 위한 제어 센터입니다.

`Open editor` 키바인드로 엽니다.

## 레이아웃

- **왼쪽 패널:** 작업 및 번들을 생성/편집/제거합니다.
- **오른쪽 패널:** 현재 페이지 목록(루트 또는 현재 번들).
- **왼쪽 상단 필터:** 제목, 메모, 작업 유형 텍스트별로 필터링합니다.
- **왼쪽 하단:** 가져오기/내보내기, 구성, 닫기.

## 버튼 추가

- `Add Key Action`
- `Add Command`
- `Add Item Equip`
- `Add Bundle`

## 목록 상호작용

### 생쥐

- **항목에 좌클릭:** 선택하세요.
- **LMB 드래그 항목:** 현재 페이지 내에서 재정렬됩니다.
- **LMB를 사용하여 항목을 번들 행으로 드래그:** 항목을 해당 번들로 이동합니다.
- **LMB를 사용하여 항목을 뒷줄로 드래그:** 항목을 상위/루트로 이동합니다.
- **번들 행의 RMB:** 해당 번들을 엽니다.

### 키보드

- `Ctrl + F`은 필터 상자에 초점을 맞춥니다.
- `Enter`은 선택된 행을 편집합니다.
- `Delete` 또는 `Backspace`은 선택한 행을 제거합니다.
- `Up Arrow` 선택한 항목을 위로 이동시킵니다.
- `Down Arrow` 선택한 항목을 아래로 이동합니다.

!!! 팁
    모호한 재정렬을 방지하기 위해 필터 텍스트가 활성화된 동안 키보드 위/아래 이동은 비활성화됩니다.

## 표시되는 행 유형

- 일반 아이템 행(액션 또는 번들)
- 탐색경로 행(`root/.../bundle`)
- `Back to root`
- `Back to <parent>`

## 잠긴 항목

잠긴 항목은 게임 내 삭제 작업으로부터 보호됩니다.

- GUI 삭제로는 제거할 수 없습니다.
- 잠금을 적용하는 API 제거 호출로는 제거할 수 없습니다.
- `config/ezactions/menu.json`을 수동으로 편집하여 제거할 수 있습니다.

## 빠른 편집을 위한 팁

- 깔끔한 방사형 라벨을 위해 짧은 제목을 사용하세요.
- 공유 유틸리티 작업을 번들로 묶습니다(루트 혼란을 줄이기 위해).
- 메뉴가 커질 때 필터 상자를 빠른 유형 검색으로 사용하세요.

???+ info "심층 분석: 드래그 앤 드롭 동작"
    - 재정렬은 삽입 논리를 사용합니다(간단한 교환이 아님).
    - 번들에 추가하면 해당 번들의 하위 목록에 추가됩니다.
    - 상위/루트로 드롭하면 자동 점프 대신 현재 편집기 보기가 유지됩니다.
    - 성공한 모든 이동/제거 작업은 즉시 디스크에 유지됩니다.
