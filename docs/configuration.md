# Configuration

EZActions provides a configuration screen accessible from the Menu Editor under **Settings**.

All settings are saved per profile and apply client-side only.

---

## General Settings

| Option | Description | Default |
|---------|--------------|----------|
| Move While Radial Open | Allows the player to move while the radial menu is open. | true |
| Show Tooltips | Displays notes when hovering over actions. | true |
| Smooth Open/Close | Enables the radialâ€™s animation when opening. | true |

---

## Visual Settings

| Option | Description | Default |
|---------|--------------|----------|
| Ring Thickness | Controls the thickness of radial segments. | 0.7 |
| Size Multiplier | Scales the radial overall. | 1.0 |
| Hover Grow % | Enlarges hovered items slightly. | 10 |
| Blur Background | Disables background blur (controlled by mixin). | false |

---

## Animation Settings

| Option | Description | Default |
|---------|--------------|----------|
| Open Duration | Animation time for opening the menu (ms). | 150 |
| Close Duration | Animation time for closing (ms). | 120 |
| Easing Curve | The interpolation curve for animations. | Smooth In-Out |

---

## Color Customization

- **Item Highlight:** Adjust highlight color for hovered items.
- **Outline:** Toggle outline rendering.
- **Background Tint:** Controls transparency and shade of radial background.

---

## Resetting Configuration

You can reset all configuration options from the Settings screen by clicking **Reset to Defaults**.  
Changes apply immediately and persist between sessions.
